# Select a feature

Use the interact plugin to add feature selection to your map. This example adds a plain dataset of field parcels (see [Add polygons](/examples/add-polygons)) and the ability to select one to display its properties in a panel. Selecting another parcel replaces the current selection; clicking outside clears it.

:::note
The layer ID used here (`field-parcels`) comes from the datasets plugin — a dataset with no sublayers gets a layer ID matching its own `id`. For other layers, including ones from your base map style, set `debug: true` in the plugin options, open the browser console, and click the map to see which layer IDs are available at that point.
:::
